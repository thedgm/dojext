Dojext

dojo boilerplate for chrome extension

Download this project as a .zip file Download this project as a tar.gz file

Wecome to Dojext.

This is a boilerplate for Dojo powered Google Chrome extension.

The extension consists of three pages:

  • background page
  • main tab page
  • options page

Just enough to load in your AMDs for a quick spin!

Content security policy in manifest.json lets you use Google CDN. This saves at least 11MB of your project space.

Dodo, Dijit and DojoX ready for you.

Options page is a sample page, styled with Dijit Theme - Claro.

Enjoy!
